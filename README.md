# RuleGuard: Enhancing Reliability in LLM-Based Smart Home Systems

## Project Type

Team Project (5 Members)

## Overview

RuleGuard is a hallucination-aware smart home automation framework designed to improve the reliability and safety of Large Language Models (LLMs) in IoT environments.

The framework converts natural language commands into executable smart home actions while detecting and mitigating hallucinated outputs through knowledge-based validation and correction mechanisms.

---

## Problem Statement

Large Language Models can generate incorrect or unsafe smart home commands, including:

- Non-existent devices
- Invalid room assignments
- Unsupported actions
- Ambiguous user requests

These issues reduce reliability and trust in smart home automation systems.

---

## Solution

RuleGuard implements a five-stage pipeline:

1. Command Understanding
2. Rule Generation
3. Hallucination Validation
4. Mitigation Engine
5. Execution Simulation

The system validates generated commands against a smart home knowledge base before execution, ensuring safer and more reliable automation.
